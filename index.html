Our Chrome extension reads your ERP attendance page and fills all subjects into the calculator automatically. One click — done.

Download Extension Works on Chrome & Edge
Installation steps
1
Download & unzip
Click the Download button above. Once downloaded, right-click the zip → Extract All → extract to a folder you'll remember.
2
Open Chrome Extensions
In Chrome, type chrome://extensions in the address bar and press Enter.
3
Turn on Developer Mode
Toggle Developer mode on — it's in the top right corner of the Extensions page.
4
Load the extension
Click Load unpacked → navigate into the unzipped folder → select the inner kl-extension folder that contains manifest.json → click Select Folder.
5
Open ERP & search
Go to newerp.kluniversity.in → log in → click Attendance Register → select year & semester → press Search.
6
Sync to calculator ⚡
Click the extension icon in your Chrome toolbar → your subjects appear → click Send to Calculator → done! All subjects auto-fill instantly.
⚠ Common issues
No data found? Make sure you pressed Search on the ERP page first.
Wrong folder selected? Select the inner folder that has manifest.json inside it.
Extension not showing? Click the puzzle piece 🧩 icon in Chrome toolbar to pin it.
